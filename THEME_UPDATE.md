# Alshaya Enterprises Co-Estimate - Theme Update

## Application Rebranding

### Name Change
- **Previous Name**: Questemate
- **New Name**: Alshaya Enterprises Co-Estimate

### Color Scheme Update

Inspired by the Alshaya Enterprises website (https://alshayaenterprises.com/), the application now features an elegant gold and navy blue color palette.

#### Primary Colors
- **Gold Primary**: `#d4af37` (Warm, luxurious gold)
- **Gold Accent**: `#f4d03f` (Bright gold highlights)
- **Gold Dark**: `#b8941f` (Deeper gold for gradients)

#### Secondary Colors
- **Navy Blue**: `#1a365d` (Deep navy for backgrounds)
- **Navy Dark**: `#0f1419` (Almost black for text on gold)
- **Light**: `#f8fafc` (Light text on dark backgrounds)

#### Supporting Colors
- **Success**: `#10b981` (Green - unchanged)
- **Warning**: `#f59e0b` (Orange - unchanged)
- **Danger**: `#ef4444` (Red - unchanged)
- **Info**: `#06b6d4` (Cyan - unchanged)

### Brand Consistency

The new theme maintains the professional, premium feel of Alshaya Enterprises while preserving all functionality of the AI-powered BOQ processing platform. The gold and navy combination conveys:

- **Luxury**: Gold accents suggest premium service
- **Trust**: Navy blue represents stability and professionalism
- **Innovation**: Modern gradients and animations show technological advancement
- **Clarity**: High contrast ensures excellent readability

### Technical Implementation

All color changes were implemented using:
- CSS variable updates in `:root`
- Global find-and-replace for hex color codes
- Gradient adjustments throughout the stylesheet
- Text color optimizations for accessibility

The application maintains WCAG AA contrast standards for all text elements.
