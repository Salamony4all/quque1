## Typography Scale

```
H1 (Page Title):     4em / 800 weight
H2 (Card Title):     2em / 800 weight
H3 (Section Title):  1.5em / 700 weight
Body Large:          1.2em / 400 weight
Body Normal:         1em / 400 weight
Small Text:          0.9em / 400 weight
Button Text:         1.15em / 700 weight
```

## Spacing System

```
Micro:   8px   (gaps, icon margins)
Small:   16px  (element padding)
Medium:  24px  (section spacing)
Large:   40px  (component spacing)
XL:      60px  (page section spacing)
XXL:     80px  (major divisions)
```

## Shadow Levels

```
Level 1 (Subtle):
  0 2px 8px rgba(0,0,0,0.08)

Level 2 (Card Default):
  0 15px 50px rgba(0,0,0,0.08)

Level 3 (Card Hover):
  0 20px 60px rgba(99,102,241,0.15)

Level 4 (Button Hover):
  0 15px 40px rgba(99,102,241,0.5)

Level 5 (Maximum):
  0 35px 80px rgba(0,0,0,0.3)
```
